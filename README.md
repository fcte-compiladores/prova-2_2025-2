Este repositório possui os arquivos iniciais para responder questões práticas
da Prova 2 de Compiladores (2025.2). Baixe o repositório e edite as questões no
seu computador e depois submeta via aprender apenas o conteúdo do arquivo 
especificado em cada uma das questões. 

As questões possuem testes unitários que podem ser executados localmente 
utilizando o Pytest. Veja o README de cada questão para mais detalhes.

Boa sorte!
